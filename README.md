# Jootsing Plugins

Claude Code plugins by [Jootsing Research](https://jootsing.com).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [liveactivities](./plugins/liveactivities) | iOS Live Activity notifications for Claude Code |

## Installation

Add the marketplace to Claude Code:

```
/plugin marketplace add jootsing-research/jootsing-plugins
```

Then install any plugin:

```
/plugin install liveactivities@jootsing-plugins
```
